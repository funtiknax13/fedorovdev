01

Обо мне

Python-разработчик полного цикла: требования → архитектура → код → запуск → поддержка. Начинал с Django и DRF, сейчас одинаково уверенно собираю сервисы на FastAPI и фронтенд на Vue 3. Пишу чистый SQL, проектирую модели данных, разворачиваю в Docker и сопровождаю продукт после релиза.

Мой флагманский проект — airunningcoach.pro: SaaS для бегунов на FastAPI + Vue с разбором FIT-файлов тренировок, аналитикой дистанции и темпа, целями и AI-тренером на базе OpenAI. От схемы базы данных до продакшен-деплоя в Docker — сделано в одиночку.

  • Цикл работы Требования → код → запуск → поддержка
  • Backend Python · Django · FastAPI · DRF
  • Frontend Vue 3 · TypeScript · JavaScript
  • Данные и инфра PostgreSQL · Docker · CI/CD

Что я делаю

Веб-сервисы разного уровня: от backend и API до готового SaaS с современным интерфейсом, биллингом и AI.

/01

Backend и API

Django, FastAPI, DRF. REST API, авторизация и роли, бизнес-логика, фоновые задачи (Celery, APScheduler), интеграции внешних сервисов и платежей.

Веб-приложения и SaaS под ключ

От прототипа до запуска: backend + фронтенд на Vue или Django-шаблонах, личные кабинеты, подписки, админка и деплой в Docker.

/03

AI-интеграции и автоматизация

Подключение OpenAI и LLM в продукт: ассистенты, аналитика, генерация и обработка данных. Автоматизация рутинных процессов.

/04

Доработка и поддержка

Развитие функционала, рефакторинг и сопровождение существующих проектов на Django и FastAPI.

06

Частые вопросы

На каких технологиях вы работаете?

Backend на Python: Django, DRF и FastAPI. Фронтенд на Vue 3 или Django-шаблонах. Базы данных — PostgreSQL, инфраструктура — Docker, Nginx и CI/CD. Подключаю AI через OpenAI API.

Делаете только backend или фронтенд тоже?

И то, и другое. Могу закрыть весь цикл: API и бизнес-логику на backend плюс интерфейс на Vue 3 или Django-шаблонах с базовым JavaScript.

Можете подключить AI к продукту?

Да. Интегрирую OpenAI и другие LLM — ассистенты, аналитика, обработка данных. Рабочий пример — AI-тренер в сервисе airunningcoach.pro.

Беретесь за доработку готового проекта?

Да. Развиваю, рефакторю и сопровождаю существующие проекты на Django и FastAPI.

Как оценивается стоимость?

После брифа — фиксированная стоимость по этапам. Обсуждаем прозрачно, без скрытых доплат.
